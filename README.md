# AI Travel Planner

A stunning, modern home page for an AI Travel Planner web application built with React, TailwindCSS, Framer Motion, and Three.js.

## Features

- 🎨 **Dark, cinematic design** with gradient overlays
- ✨ **Smooth animations** powered by Framer Motion
- 🌍 **3D interactive globe** using React Three Fiber
- 💫 **Glassmorphism effects** throughout the UI
- 🎯 **Parallax animations** and floating elements
- 📱 **Fully responsive** design

## Tech Stack

- React 18
- TailwindCSS
- Framer Motion
- Three.js / React Three Fiber
- Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx      # Sticky glassmorphic navigation
│   ├── Hero.jsx        # Hero section with 3D globe and CTA
│   ├── Features.jsx    # Feature cards with animations
│   └── Footer.jsx      # Minimal footer
├── utils/
│   └── mockApi.js      # Mock API for chat integration
├── App.jsx             # Main app component
├── main.jsx            # Entry point
└── index.css           # Global styles
```

## Mock API

The project includes a mock API utility (`src/utils/mockApi.js`) that simulates chat responses. This will be replaced with the actual Gemini API integration later.
